# Optitrack and ROS 2
**Task**: obtain motion capture pose data over wifi on a ros 2 docker container.


The original `ros-XXX-mocap-optitrack` package doesn't work on ROS 2, so we will use this 3rd party package instead.
I modified the files to work off the bat with the Optitrack setup on Level 18 Drone Lab, the only thing you need to change is the object name.
